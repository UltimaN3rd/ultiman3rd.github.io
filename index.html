Nick's Saturday Morning Games

Collection of small games on Steam
November 1st 2025

This is my platform for releasing small games in a collection on Steam. They are programmed from scratch using my Croaking Kero Game Framework (see below).

Current games included:

Grapestomper Rescue: A helicopter flying game where you rescue orphaned Grapestompers from rock-throwing, gun-toting worms. Stompers will stomp grapes to make wine at your base, which you can sell to buy upgrades.
Transistor Glow: Electronics puzzle game where you use a few simple components to achieve endless complexity. Make bigger devices in Creative Mode and share them with your friends!

Source code is released under the GPL in this git repository on Codeberg.

Available now on Steam early access: https://store.steampowered.com/app/2572310/Nicks_Saturday_Morning_Games/

Croaking Kero Game Framework

Free & open source
October 1st 2025
A game backend framework, programmed from scratch for Windows, Linux and MacOS

When programming games in the past, I've listened to the advice, "Don't reinvent the wheel," and, "You probably can't do it better than X, so just use X." This advice is intended to save people time and get straight to productive game programming, but I found it difficult to become a good enough programmer while skipping so many foundational parts of the code. I ran into bugs and issues which required understanding the underlying code, which was very difficult when I'd never written code as low-level as the libraries/engines I depended on. I also felt demotivated working on games that I ultimately may be unable to maintain - like the multitude of fantastic Flash-based games from back in the day.

Source code on Codeberg and Github
